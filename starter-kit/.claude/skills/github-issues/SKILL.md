# GitHub Issues

## Minimumsskabelon

```markdown
## Beskrivelse
[Hvad skal laves og hvorfor?]

## Task List
- [ ] Implementation
- [ ] Tests
- [ ] Dokumentation

## Acceptance Criteria
- [ ] Kriterie 1
- [ ] Kriterie 2

## Definition of Done
- [ ] Relevante tests er grønne
- [ ] Code review foretaget
- [ ] Dokumentation opdateret
- [ ] PR merged til develop
```

## Regler

- Gør bugs målbare
- Skriv root cause når den er kendt
- Luk ikke issues med åbne DoD-checkboxes
